CESS Lab Remote

The Center for Experimental Social Science Lab Remote is an Open Source cross platform tool developed to remotely start, stop, and monitor a program on several computers from one location in a computer lab environment.

Installation

The CESS Lab Remote consists of two applications the Server and the Client. The server application will be installed on only one computer (the server). The Client app will be configured and copied on to all of the computers in the lab to be remotely controlled.

Server Installation

  1. Download the Server app.
  2. Unzip the Remote-Server.zip file on the computer acting as the server.
  3. Edit the application_info.txt in the Server folder and add the programs you wish to run remotely.

File Format: application_info.txt
<Program Name>, <Path To Program>, <Flags>,

Example:
Windows Media Player, C:\Program Files\Windows\Media Player\wmplayer.exe,,

Client Installation

  1. Download the Client app.
  2. Unzip the Remote-Client.zip file on each client computer you wish to run programs remotely on.
  3. Edit the client_location.txt in the Client folder and enter the servers ip address.

File format: client_location.txt
<IP address>, <Server Port>
Example:
192.168.0.1, 2600
Note: 2600 is the default port used by the server.
